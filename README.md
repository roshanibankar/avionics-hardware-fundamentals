#### Active Signal Processing
* Op-amp configuration, comparator networks, and Schmitt trigger hysteresis conditioning to eliminate state-transition jitter in high-EMI environments.

#### Sensor Interfacing
* **`Signal Conditioning/lights-audio-sync`**

#### Mixed-Signal Integrity
* ADC resolution enhancement via oversampling, decimation, and mixed-signal ground-loop noise mitigation.

#### Hardware Debouncing
* Transient electromechanical contact bounce analysis and hardware RC vs. software ISR debouncing networks.

#### Temporal Acquisition
* **`depth`**
* **`Timing and Measurement/Valentine's-heart-...`**

#### Power Electronics
* **`Valentine's-heart-pcb`**
